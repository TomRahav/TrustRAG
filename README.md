# TrustRAG

## Quick Start

### Installation

```
git clone https://github.com/HuichiZhou/TrustRAG.git

conda create -n trustrag python=3.10

conda activate trustrag

pip install lmdeploy

pip install beir

pip install nltk

pip install rouge_score

python run_trustrag.py
```
